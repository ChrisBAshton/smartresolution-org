SmartResolution

SmartResolution is Online Dispute Resolution software which is:

  • Free and open source software (FOSS), meaning anyone can install the software for any purpose.
  • Fully extensible, meaning developers can write modules that can be plugged into any installation of SmartResolution. All sorts of exciting units of functionality can be plugged into the system, making your ODR platform even more powerful. Check out the Marketplace for a full list.
  • Enterprise level, backed up by integration and unit tests, continuous integration, CodeClimate and other tools.

Ready to get started? The installation process is simple.

Live Demo

http://demo.smartresolution.org is a live demo of the SmartResolution software. Feel free to login using credentials you can find in the project's fixture data, or register your own account.
